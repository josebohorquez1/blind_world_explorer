Free explore settings

Adjust how movement and orientation behave while using free explore mode. These settings control how far you move, the direction you face, and how much you rotate when turning.

Moving distance

Set how far you move each time you move forward or backward. Choose the distance value and the unit used for movement.

Direction

Set the direction you are facing in degrees. For example, 0° represents north, 90° east, 180° south, and 270° west.

Increment rotation

Set how many degrees you rotate each time you turn left or right. Smaller values allow finer adjustments in direction.
